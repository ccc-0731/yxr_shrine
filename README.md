# 🏮 于神庙

## 🙏 这是什么？
**于神庙** 是一个基于现代前端玄学的赛博祈愿网站。  
你可以在这里：

- 🔥 给于神烧香  
- 🎋 抽签看运势  
- 📈 观察功德值随缘波动  
- 🧠 假装自己在做一件“很有仪式感但毫无科学依据”的事  

适用人群：
- 高考考生  
- 被考试支配的人  
- 对玄学抱有怀疑但仍想试试的人  
- 想整活给朋友看的人  

---

## 🏯 核心功能

### 🔥 烧香（点香）
- 每点一次香：
  - 香火 +1  
  - 功德值随机 +2 ~ +7  
  - 于神随机回应一句（是否有用，未知）
- 连点过快可能被于神无视（冷却机制）

> 于神：香已收到，正在后台排队处理……

---

### 🎋 抽签
- 抽签等级包含：
  - **大吉**
  - **中吉**
  - **小吉**
  - **抽象**
  - **寄**
- 抽到什么完全取决于：
  - 运气  
  - 心态  
  - JavaScript 的 `Math.random()`

> 抽象签 ≠ 坏  
> 寄签 ≠ 寄了  
> 一切解释权归于神所有

---

### 📊 功德 / 祝福值
- 范围：0–100  
- 可升可降，起伏不定  
- 刷新页面不会消失（localStorage 保佑）
